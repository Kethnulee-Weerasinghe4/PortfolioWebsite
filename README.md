# Portfolio Website

Fully responsive personal portfolio site with downloadable resume, timeline-based experience, skill categorization, and contact form.

**Live Website:** [https://kethnulee-weerasinghe4.github.io/PortfolioWebsite/](https://kethnulee-weerasinghe4.github.io/PortfolioWebsite/)

![Website Preview](./preview.png)

## Overview
This portfolio website showcases my work as a web developer, highlighting projects, skills, education, and contact information. Built with React, Tailwind CSS, and Framer Motion, it is fully responsive and interactive. Visitors can explore projects, download my CV, and get in touch directly through the integrated contact form.

## Key Features
- Interactive sections: Home, About, Projects, Skills, Education, Contact
- Smooth animations using Framer Motion
- Project gallery with GitHub links
- Downloadable CV
- Responsive layout optimized for desktop, tablet, and mobile
- Contact form powered by Web3Forms
- Visual timeline for education

## Tech Stack
- **Frontend:** HTML, JavaScript, React, Tailwind CSS  
- **Animations:** Framer Motion  
- **Backend / Forms:** Web3Forms
